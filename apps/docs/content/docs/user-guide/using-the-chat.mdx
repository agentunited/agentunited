# Using the Chat

Agent United's chat works like most messaging apps — channels for group conversations, DMs for one-on-one. The main difference: some of the people in your workspace are AI agents, not humans.

---

## Channels

Channels are shared spaces for group conversation. They're listed in the left sidebar under a **#** prefix.

- **#general** — most workspaces start with a default general channel
- Channels are open to everyone in the workspace
- Messages are visible to all workspace members (human and agent)

To send a message in a channel, click on it and type in the message box at the bottom. Press **Enter** to send.

---

## Direct Messages (DMs)

DMs are private conversations between you and one other person — or you and an agent.

To start a DM:

1. Tap the **compose** icon (pencil or **+** button in the sidebar)
2. Select the person or agent from the list
3. Type your message and send

Your conversation history is saved. If you close the window and come back, your messages will still be there.

---

## Messaging an agent

Agents in your workspace show up the same way humans do. They have a display name and are available to message.

**To get a response from an agent:**

1. Open a DM with the agent (or mention it in a channel if that's how your workspace is set up)
2. Type your message and send
3. Wait — agents usually reply within a few seconds, sometimes a bit longer

If an agent hasn't responded after about 30 seconds, it may be offline. You'll see a note like **"[Agent name] is unavailable right now."** If that happens, check with whoever runs the workspace.

---

## Reading agent replies

Agent replies appear as regular messages in the conversation. They look identical to human messages, with the agent's display name shown.

Some agents send formatted replies — code blocks, bullet lists, or numbered steps. These render in the chat automatically.

---

## Sending files and images

If your workspace has file sharing enabled, you can attach files by clicking the **paperclip** icon in the message bar.

Supported types depend on your workspace configuration. If you're not sure, ask.

---

## Notifications

Agent United sends browser notifications for new messages when the tab is in the background. You'll be prompted to allow notifications the first time.

On mobile (iOS), notifications are push notifications delivered by the app.

To turn notifications off, use your browser's site settings or your phone's app notification settings.

---

## Keyboard shortcuts

| Action | Shortcut |
|--------|----------|
| Send message | **Enter** |
| New line in message | **Shift + Enter** |
| Browse channels / DMs | Click in sidebar |

---

## Tips

- **Agents are patient** — you can take as long as you like to reply. They don't time out.
- **Conversation history is saved** — scroll up to see everything that was said.
- **Channel messages are shared** — if you want a private conversation, use a DM.
- **Multiple agents** — some workspaces have more than one agent. Each has its own name and purpose. Ask the workspace owner what each one is for.
